Programmer une carte NFC

Génère l'URL à écrire dans NFC Tools pour une salle.

Procédure NFC

Écriture directe Android

Sur Chrome Android compatible, SAC peut écrire directement l'URL dans une carte NDEF. Appuyez sur le bouton d'écriture, acceptez la permission NFC, puis approchez la carte du téléphone.

Le bouton “Verrouiller définitivement” passe la carte en lecture seule permanente. Pour un verrouillage par mot de passe réversible, utilisez un outil natif compatible avec le modèle exact du tag.

Écriture directe indisponible

Ce navigateur ne permet pas l'écriture NFC depuis une PWA. Utilisez NFC Tools sur iOS, ordinateur, ou navigateur Android non compatible.

Écriture

  1. Installer l'application NFC Tools depuis l'App Store ou le Play Store.
  2. Ouvrir NFC Tools, onglet Écrire, puis Ajouter un enregistrement.
  3. Choisir URL / URI et coller l'URL générée pour la salle.
  4. Appuyer sur Écrire et présenter la carte NFC au téléphone.
  5. Tester immédiatement la carte: elle doit ouvrir SAC avec le paramètre nfc, puis l'URL doit être nettoyée par l'interface.

Verrouillage définitif

  1. Ne verrouiller qu'après un test réussi sur au moins un téléphone iOS/Android.
  2. Dans NFC Tools, utiliser la fonction de verrouillage / lecture seule de la carte uniquement pour les cartes de production finales.
  3. Confirmer uniquement si l'URL est correcte: l'opération est irréversible sur la plupart des tags.
  4. Étiqueter physiquement la carte avec la salle et conserver des cartes vierges de secours.
Une carte doit contenir uniquement l'URL SAC officielle de la salle, par exemple https://domaine.tld/?nfc=room_uid.
